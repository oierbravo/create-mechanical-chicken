# Create Mechanical Chicken

Addon for create.
Generates eggs with rotation and fluid.

## Features

- Mechanical Chicken Kinectic block.
- Ponder
- Compat: JEI & JADE
- Configurable:
    - Processing time
    - Output amount
    - Stress impact
    - Fluid capacity
    - Required fluid amount
    - Required fluid type
